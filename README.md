# Boon Bans

Control which blessings can be offered by gods, NPCs, hammers, and keepsake-style sources.

Part of the [Run Director modpack](https://thunderstore.io/c/hades-ii/p/adamant/RunDirector_Modpack).

## What It Does

Boon Bans lets you trim, bias, and reshape the boon offer pool across multiple sources.

The module is split into four main scopes:

- Olympians
  Control offerings from the nine core Olympians.
- Other Gods
  Control offerings from bonus gods such as Hermes, Artemis, and Athena.
- Hammers
  Control hammer offerings for each weapon.
- NPCs
  Control offerings from NPCs such as Arachne, Narcissus, Echo, Hades, Medea, Circe, and Dionysus. It also controls which Hades boon can be offered by Jeweled Pom.

Within those scopes, the module can:

- ban individual boons so they stop appearing from that source
- force rarity behavior for eligible boons

It also includes run-level settings for:

- Improve First N Boon Rarity
  Forces the first several boon offers to roll at higher rarity.
- Bridal Glow target selection
  Lets you point that effect at a specific boon target where supported.

Use it when you want more control than broad god-pool filtering gives you. God Pool controls which gods can enter the run; Boon Bans controls what those sources can offer once they are active.

## Installation

Install via r2modman.

This module is usually installed as part of the full [Run Director modpack](https://thunderstore.io/c/hades-ii/p/adamant/RunDirector_Modpack), where it appears in the shared Run Director UI with the other run-control modules.
